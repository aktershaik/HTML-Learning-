CSS UNIT

  1. width: 500px;
  2. margin: 50px;
  3. padding: 20px
  4. font-size: 30px

There are Two types of Unit in CSS

  1. Absolute
    1. cm (1 cm = 37.7px)
    2. mm (1 mm = 3.7px)
    3. inc(1 inch = 96px)
    4. px
    5. pt (1 Inch = 72pt)
    6. pc (1pc = 12pt)
  2. Relative
    1. em = relative to font size of element
    2. rem = relative to font-size of the root element
    3. vh =viewport height
    4. vw = viewport device-width
    5. % = percemtage relative to the root element percentage
    6. pc (1pc = 12pt)
